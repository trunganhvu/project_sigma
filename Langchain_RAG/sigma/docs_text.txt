The Text function converts values to text strings (or WKT format when converting geography data).
For more information about converting the geography data type to WKT, see Text (geography).
Syntax
Text(input)
Function arguments:
input
(required) The text, date, or number data that must be converted to text
Sigma converts dates to text as a full ISO date-timestamp.
