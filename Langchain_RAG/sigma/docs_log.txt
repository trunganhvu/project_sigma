Log
Suggest Edits
Compute the logarithm of a number, defaulting to base 10.
Usage
Log(value, base)
value (required) The number for which to calculate the logarithm.
base (optional) The optional base for the logarithm. Defaults to base 10.
Example
Log(10)
Returns 1
Log(1, 2)
Returns 0
