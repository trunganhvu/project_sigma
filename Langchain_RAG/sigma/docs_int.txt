Int
Suggest Edits
Rounds the input number down to the largest integer of equal or lesser value.
Usage
Int(number)
number (required) The number to be rounded.
Example
(1) Returns 3.
Int(3.6)
(2) Returns -4.
Int(-3.2)
(3) Returns the Int for each row in the input column.
